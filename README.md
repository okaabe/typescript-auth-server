Simple Authentication server, its purpose is to start to learn typescript, prisma, axios and a little bit about JWT and web authentication

### User guide

```
- git clone https://github.com/tosuki/typescript-auth-server ./server
- cd server
- npm install
- npm build
- npm start
```
